## 📊 如何使用这些模板

1. **找到相似场景**：从上面找一个最接近你消费场景的模板
2. **参考评分**：根据模板的参考值进行调整
3. **个性化修改**：根据你的实际情况微调（±1-2分）
4. **输入工具**：在NVR工具中输入调整后的分数

## ⚠️ 注意事项

1. **模板仅供参考**：每个人的情况不同，不要完全照搬
2. **定期更新**：需求会随人生阶段变化，每季度重新评估
3. **诚实评分**：不要高估高层次需求（连接、成长、意义）
4. **对比验证**：用模板分析3-5次后，会找到自己的模式
